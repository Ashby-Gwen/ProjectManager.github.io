Task Management

Instructions:

  1. Enter your password to access editing tasks.
  2. Add a new task by typing the task name and choosing a deadline, then click Add Task.
  3. Update a task’s status by checking the box next to it. Completed tasks will cross out automatically.
  4. Export your updated task list anytime by clicking Export CSV. You can upload this CSV to GitHub if needed.
